This directory contains a distribution of The GNU Project Debugger (GDB) in object form.
The binary was cross-compiled for Windows with MinGW-w64, and is statically linked against libgcc and libstdc++.
This distribution of GDB is configured for debugging remote Linux applications from a local Windows system.

The licenses for GDB and its dependencies can be found in the `licenses` subdirectory.

The source code for GDB and its dependencies can be downloaded from the following URLs:

- https://ftp.gnu.org/gnu/gdb/gdb-11.2.tar.gz
- https://gmplib.org/download/gmp/gmp-6.2.1.tar.xz
- https://github.com/gcc-mirror/gcc
